Setup Instructions

To enable the floating AQI widget, you need to install the AQI Widget app:

  1. Download the app: Download AQI Widget
  2. Open the downloaded file to install the app. You may need to enable "Install from unknown sources" in your settings.
  3. Return to this page and click "Install Widget" again.
